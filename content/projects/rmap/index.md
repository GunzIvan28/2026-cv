rMAP, the Rapid Microbial Analysis Pipeline, is an easy-to-use resistome profiling bioinformatics pipeline for ESKAPE pathogens using Illumina paired-end whole-genome sequencing reads.

The workflow supports analysis of Enterococcus faecium, Staphylococcus aureus, Klebsiella pneumoniae, Acinetobacter baumannii, Pseudomonas aeruginosa, and Enterobacter species.
